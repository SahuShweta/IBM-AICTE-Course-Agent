## IBM SkillsBuild AICTE Internship 2026

Agentic AI for Personalized Course Pathways

---

# 📌 Project Overview

The AI Personalized Course Pathway Agent is an AI-powered web application that helps students generate personalized learning roadmaps based on their interests, skill levels, and career goals.

The system analyzes user input and recommends:
- Skills to learn
- Technologies to study
- Project ideas
- Career preparation roadmap
- Certifications

This project aims to simplify career planning and help students choose the right learning pathway efficiently.

---

# 🚀 Features

- Personalized roadmap generation
- Dynamic skill recommendations
- Career guidance
- Project recommendations
- User-friendly interface
- Beginner to advanced pathway support

---

# 🛠 Technologies Used

- Python
- Streamlit
- IBM Granite
- IBM Cloud Lite
- LangFlow

---

# 🧠 Agentic AI Workflow

The system works using multiple AI-inspired modules:

1. Skill Analysis Agent  
   - Identifies user skill level

2. Roadmap Recommendation Agent  
   - Generates personalized learning roadmap

3. Project Recommendation Agent  
   - Suggests projects according to user interest
